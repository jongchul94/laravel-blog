# Laravel Blog with Railway

## 소개
간단한 사용자 인증 기반 블로그 애플리케이션입니다.

## 주요 기능
- 사용자 회원가입 / 로그인
- 게시글 / 댓글 기능
- 관리자 권한
- 검색, 다크 모드 지원
